Notes: 
For seek, flee, pursue, evade, arrive, and flocking, the blue agent is the target, which can be controlled with WASD.
The number of AI agents displayed can be adjusted with the up and down arrow keys.
To switch between different steering methods, press the number key associated with each one. (Seek: 1, Flee: 2, Pursue: 3, Evade: 4, Arrive: 5, Wander: 6, Path Following: 7, Separation: 8, Collision Avoidance: 9, Obstacle Avoidance: 0). Flocking is toggled by pressing shift.
Pressing enter will randomize the positions and orientations of all the AI agents.
In Obstacle Avoidance, pressing space will randomize the positions, angles, and lengths of the walls.
